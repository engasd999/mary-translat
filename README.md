# مترجم الفيديوهات

تطبيق أندرويد لترجمة الفيديوهات تلقائياً باستخدام:
- ML Kit لقراءة النصوص
- FFmpeg لمعالجة الفيديو
- LibreTranslate API للترجمة

## المتطلبات
- Android Studio 2022+
- JDK 11
